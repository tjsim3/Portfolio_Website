TI-84 Kinematics

Made with Python

Project Overview

Key Features

  • Solves kinematic equations using Python on a computer, or a calculator
  • Has larger and smaller versions to fit space needed

Technologies Used

The code was developed using Python on VScode

How I incorporated AI into this project

In the first iteration (run on the computer) I wanted to have the most possible options to solve for variables, so I used chatGPT to write a list with the all possible equations derived to solve for specific variables when given others. This would have taken an hour or two of my time, and AI was able to complete it in a few seconds. I also used AI to explain to me what I was doing wrong when I had errors, as when I was making this project I was still learning Python.

Challenges Faced

The only challenge I faced on this challenge was my lack of Python knowledge. After making this project, I begun watching the MIT coursework YouTube videos, and begun lessons on Codedex.io.

Future Improvements

In the future, I may improve this to be more easy to read, as right now it is fairly hard to comprehend since the text is so small on the calculator.

Impact

This program made it easier for me and friends to solve kinematic equations quickly when doing homework
