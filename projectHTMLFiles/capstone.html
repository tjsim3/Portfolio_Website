Poetic Chatbot

Learning with Udemy

Project Overview

A small chatbot programmed to respond to user input with poetry. Get an openai api key to use the program. This is my first AI project, so it is very limited. The example prompt in the code is the question "Are you happy?" in which the model responds in a poetic manner as instructed.

Key Features

  • Responds to user input with poetry
  • Utilizes the OpenAI API for text generation
  • It's name is Lemmy

Technologies Used

Jupyter Notebook, OpenAI API

Challenges Faced

As this was my first AI project, I faced challenges in understanding how to effectively use the OpenAI API and structure the chatbot's responses, however the 365 data science AI bootcamp course on udemy walked me through the process

Impact

This project was a kickstart to more AI projects in the future.
