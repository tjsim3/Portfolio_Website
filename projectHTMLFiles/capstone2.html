Big Data Analysis

Coded using Jupyter Notebook

Project Overview

A project I did in my work-based-learning class to display my new data analysis skills. This was a combination of what I learned from the Artificial Intelligence course I took on Udemy and the curriculum from my AP Statistics course. This includes me using a random forest model that copilot taught me how to use, and multiple graphs and tests to analyze and visualize the data.

Key Features

  • Utilizes a random forest model for predictions
  • Includes multiple visualizations and usage of statistical tests to represent the data
  • Compares large data sets
  • Is a complete mess but that's ok

Technologies Used

Jupyter Notebook

Challenges Faced

At first I had copilot guide me on how to begin this project, and had it suggest ML functions I could use to predict if the data has any correlation, and it suggested the random forest regressor that I did not know and had to learn. In addition, I used the wrong variable at the start and switched to the correct one after realizing my mistake. I also struggled to download the data from the internet as they came in different formats that I learned how to work with.

Impact

This project was a display of what I know about statistics and python, and how I can apply machine learning techniques to real-world data analysis problems.
